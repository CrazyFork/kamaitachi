About kamaitachi

kamaitachiは学習用にフルスクラッチで書かれたブラウザです。Parse、Style(Render Treeの構築)、Layout、Paintの処理を一部ライブラリを使っていますが、基本的にはフルスクラッチで実装しています。 言語はRustを使って実装されており、Servoを参考に実装を進めています。

ここではkamaitachiがどのように処理されているのかを簡単に説明します。

Parse

Parse処理では、HTMLファイルに書かれているプログラムをDOMにparseしたり、CSSをCSSOMへとparseします。DOMやCSSOMはファイルに書かれている、tagやattribute、selectorやpropertyなどのプログラムをTree構造にまとめていく作業です。 この処理を行うことでStyle処理でDOMとCSSを合体させやすくします。

Style(Render Treeの構築)

Render Treeの構築では、DOMやCSSOMをtraverseして、各Nodeに一致するCSSルールを適用していきます。

Layout

Layout処理では、Render Treeをdisplay propertyによって各Boxに分類したり、各Nodeの正確なpositionを決定していきます。

Boxというのは、display: block;とdisplay: inline;では配置の方法が違うので、displayごとにboxに分けて、それぞれのpositionを決定しやすくします。

Paint

Paint処理では、Layout処理で決定したpositionを基に実際にGUIとしてレンダリングしていきます。
